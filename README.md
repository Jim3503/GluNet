# GluNet: Dual-Modality Latent Gloss Alignment for Gloss-free Sign Language Translation

Official implementation of **GluNet**, a **gloss-free** Sign Language Translation (SLT) framework that induces **latent gloss** representations via fine-grained video–text alignment.

> Paper: *Dual-Modality Latent Gloss Alignment for Gloss-free Sign Language Translation* (PAKDD 2026, Main Track)  
> Paper ID: 151  
> Authors: Yunlong Yu, Ming Ji, Zhao Wang, Ziqian Lu

## ✨ Highlights

- **Gloss-free SLT**: no manual gloss annotations required.
- **Latent gloss induction**: discovers implicit semantic units from
  - temporally coherent **video segments**, and
  - semantically aligned **text subword tokens**.
- **Dual-modality contrastive learning** for fine-grained cross-modal alignment.
- Strong results on public SLT benchmarks (e.g., Phoenix-2014T).
